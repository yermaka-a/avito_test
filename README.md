# avito тестовое задание

### Описание задачи
Необходимо реализовать сервис, который назначает ревьюеров на PR из команды автора, позволяет выполнять переназначение ревьюверов и получать список PR’ов, назначенных конкретному пользователю, а также управлять командами и активностью пользователей. После merge PR изменение состава ревьюверов запрещено.

### Запуск проекта
1. Скачать проект с данного репозитория
2. Запустить проект командой `docker compose up`
3. Дождаться запуска БД и миграций
4. Приложение стартует на порту `8080`. БД на порту `5432`
* Все дополнительные настройки есть в `.env` 
Пароль БД по умолчанию `test`, имя бд:`test`, пользователь: `test`

### Миграции
* Миграции накатываются через docker автоматически, но есть возможность и через Makefile командами `goose-up` | `goose-down` | `dt0`
* Можно создать миграцию ч/з `migration my_migration` 

### docker-compose ч/з Makefile
* docker ч/з Make `up` | `down` | `clean`

### app ч/з Makefile
* Проект стартует ч/з docker, но можно и ч/з Make `make run` (использует `godotenv`), так же нужно учесть значение `POSTGRES_HOST` в `.env` по умолчанию = `db` как в docker-compose, для запуска напрямую нужно выставить localhost
